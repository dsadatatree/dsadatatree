Quem somos

O programa foi desenvolvido por uma equipe de três grandes profissionais, sendo um mento e dois mentorados.

Mentoria

Programa de Mentoria entre alunos DSA

Banco de Dados

O conjunto de dados adiquirido para este programa foi retirado do SINE Aberto, uma plataforma de dados dos trabalhadores.

Pré Processamento

Iremos trabalhar com o processo de pré-processamento dos dados, manipulação de malores missing, enriquecimento de dados, limpreza da valores missing e engenharia de atributos

Análise Exploratória

Demonstraremos detalhadamente o conjunto de dados, transformando os dados, em informações riquíssimas, apresentando deversos insides de nosso conjunto de dados.

Apresentação

Aqui será demonstrado uma apresentação do nosso projeto, a etapa realizada dentro do programa de mentoria e as etapas a serem desenvolvidas após o programa de mentoria.

Sumário Executivo

Iremos demonstrar o Sumário Executivo do projeto, tendo como objetivo resumir os principais tópicos do documento. Trata-se, portanto de uma breve contextualização de cada seção do projeto de mentoria, de modo a proporcionar uma visão geral do trabalho desenvolvido e de sua viabilidade.
