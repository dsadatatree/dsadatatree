Quem somos

O programa foi desenvolvido por uma equipe de três grandes profissionais, um mento e dois mentorados.

Mentoria

Programa de Mentoria entre alunos DSA

Banco de Dados

O conjunto de dados adiquirido para este programa foi retirado do SINE Aberto, uma plataforma de dados dos trabalhadores.

Pré Processamento

Iremos trabalhar com o processo de pré-processamento dos dados, manipulação de malores missing, enriquecimento de dados, limpreza da valores missing e engenharia de atributos

Análise Exploratória

Demonstraremos detalhadamente o conjunto de dados, transformando os dados, em informações riquíssimas, apresentando deversos insides de nosso conjunto de dados.

Análise Preditivo

O modelo foi desenvolvido com iteligência artificar, dando como resposta uma média de remuneração devido a sua qualificação.
